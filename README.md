# codefu-core

Claude Code skills that turn you from a **vibe coder** into an **agentic engineer** — atomic commits where every commit is one complete conceptual unit of work, and a [Linear](https://linear.app) workflow that lets you plan work, create issues, track progress, and ship features through a structured product management cycle — all without leaving the terminal.

![CODEFU!](designs/codefu-poster.jpg)

## Install

```bash
npx github:webventurer/codefu-core
```

## Documentation

```bash
pnpm dev
```

Browse the full docs at `http://localhost:5173` — getting started, install guide, skill references for `/commit` and `/linear`, and more.
